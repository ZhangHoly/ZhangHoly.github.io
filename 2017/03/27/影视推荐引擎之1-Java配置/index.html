影视推荐引擎之1-Java配置

1 Java环境配置

在如下地址找到下载地址
http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

下载地址为：

http://download.oracle.com/otn-pub/java/jdk/8u121-b13/e9e7ea248e2c4826b92b3f075a80e441/jdk-8u121-linux-x64.tar.gz?AuthParam=1490596117_22b1e061f5f544944895b1daa1b99746

新建java环境

sudo mkdir /usr/lib/jvm

java 放到开发环境里面

sudo tar -xzvf jdk-8u121-linux-x64.tar.gz -C /usr/lib/jvm/

配置java环境

sudo vim /etc/profile

尾部加入

export JAVA_HOME=/usr/lib/jvm/jdk
export CLASSPATH=.:$JAVA_HOME/lib:$JAVA_HOME/jre/lib:$CLASSPATH
export PATH=$JAVA_HOME/bin:$JAVA_HOME/jre/bin:$PATH

使能java环境

source /etc/profile

测试java环境

java -version
反馈如下就是正常
